## RUN TERMUX

```bash
apt update && apt upgrade
apt install git -y
apt install nodejs -y
apt install ffmpeg -y
termux-setup-storage
cd /sdcard
cp -r Lolichu-Bot
cd
cd Lolichu-Bot
npm install
npm start
```

## RECOMMENDED INSTALL ON TERMUX

```bash
pkg install yarn
yarn
```

## Installing
```bash
$ node .
```
